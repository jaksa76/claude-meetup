# User Stories

## Citizen – Browsing & Engagement

0001 - As a citizen I want to see all reported issues on a map as the primary view so that I can quickly understand what problems exist in my neighbourhood
0002 - As a citizen I want to tap a map pin to see the photo, description, status, and vote count of an issue so that I can learn more without leaving the map ✓

## Citizen – Reporting

0003 - As a citizen I want to take a photo of a problem so that I can visually document the issue I am reporting ✓
0004 - As a citizen I want my GPS coordinates captured automatically when I report an issue so that the exact location is recorded without manual input ✓
0005 - As a citizen I want to add an optional text description to my report so that I can provide context beyond the photo ✓
0006 - As a citizen I want to enter my phone number when submitting a report so that I can receive an SMS when the status of my issue changes ✓
0007 - As a citizen I want to upvote an existing issue so that municipality staff can prioritise the problems that matter most to the community ✓
0008 - As a citizen I want to filter map pins by category (road, lighting, waste, etc.) so that I can focus on issues relevant to me


## Municipality Employee – Issue Management

0009 - As a municipality employee I want to log in with my staff account so that I can access the internal management tools ✓
0010 - As a municipality employee I want to view a dashboard listing all open issues sorted by vote count so that I can prioritise the most pressing reports
0011 - As a municipality employee I want to change the status of an issue (New → In Progress → Resolved) so that citizens are kept informed about progress
0012 - As a municipality employee I want to assign an issue to a specific department or colleague so that responsibility is clearly allocated
0013 - As a municipality employee I want to add an internal note to an issue so that I can communicate context to colleagues without it being visible to the public

## Administrator – Account Management

0014 - As an administrator I want to create municipality employee accounts so that staff can access the management dashboard
0015 - As an administrator I want to deactivate or remove employee accounts so that former staff no longer have access
0016 - As an administrator I want to assign roles to employees (e.g. viewer, editor) so that access permissions reflect each person's responsibilities

## Suggestions

0017 - As a citizen I want to receive a unique reference number when I submit a report so that I can track its progress or reference it in future communication
0018 - As a citizen I want to see a heatmap overlay of issue density on the map so that I can identify the most problem-affected areas of the city
0019 - As a municipality employee I want to export a filtered list of issues to CSV so that I can include data in official reports
0020 - As a municipality employee I want to see a chart of issues by category and status on the dashboard so that I can spot trends at a glance
0021 - As an administrator I want to define and manage issue categories so that reports are consistently classified without code changes
0022 - As a citizen I want the app to work offline and sync my report when connectivity is restored so that I can submit issues in areas with poor signal
